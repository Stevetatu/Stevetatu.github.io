About Me

Hello! I'm Steven OChieng, a Kenyan Full Stack Developer with over 4 years of experience in software development. I specialize in backend engineering, building secure, scalable, and high-performance web applications using Python and Django. My expertise lies in designing robust APIs, architecting database systems, optimizing performance, and developing AI-powered solutions that power modern web platforms. I have extensive experience working with relational databases, authentication systems, background task processing, and integrating third-party services. While I’m comfortable across the full stack with Vue.js/Nuxt.js, my core strength is in backend architecture, system design, and building reliable server-side applications that scale. I focus on writing clean, maintainable code and delivering production-ready systems that solve real-world problems.

I've successfully built and launched multiple production applications, including scratcho-africa, a mobile application connecting Kenyan basketball fans, teams, and players through interactive community features. My expertise spans the entire software development lifecycle from backend architecture and database design to API development, deployment, and long-term system maintenance with a strong focus on building scalable and reliable backend systems

Python has been my primary programming language since 2022, and I've built most of my projects using Django as the backend framework. I'm proficient in Django REST Framework for API development, and I use modern frontend technologies like Nuxt.js, and Tailwind CSS to create responsive user interfaces.

I am a Bachelor of Science student in Mathematics and Computer Science at Kenyatta University. Beyond development, I’m deeply passionate about AI integration, having worked extensively with OpenAI GPT-4 and Claude APIs to build intelligent, agentic systems.
